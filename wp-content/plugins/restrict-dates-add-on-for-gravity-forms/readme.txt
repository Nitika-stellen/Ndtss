A simple and nice plugin to add date restrict dynamically in gravity forms default date picker field.

== Description ==
A simple and nice plugin to add date restrict dynamically in gravity forms default date picker field. If the installation is okay, go to gravity forms dashboard. Add a date field on your form. Open date field and you will see to Enable Restrict date options checkbox. Click that checkbox and configure as you want. And save form. You're done!

== Features ==
* Minimum date set to a current and specific date
* Maximum date set to a current and specific date
* Disable manually input (read-only option)
* Week/Off day disable

== Frequently Asked Questions ==

= Can I disable specific days of the week, like weekends? =
Yes. You can disable any weekday (for example, Saturday and Sunday) to prevent users from choosing those dates.

= Can I set a minimum and maximum date range? =
Absolutely. You can define both a minimum and maximum selectable date. For example, allow only dates from “tomorrow” to “30 days from today.”

= Does it support multiple date fields in the same form? =
Yes. Each date field can have its own restriction settings — all managed individually within the Gravity Forms editor.

= Can I restrict dates globally for all forms? =
Currently, restrictions are applied per field of gravity forms.

= Does it support inline date picker? =
Yes, Its supproted but only in the pro version.
